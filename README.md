# Samsung Internship Assessment – Android Developer

## Overview
This repository contains all tasks completed for the Samsung Internship position.

## Tasks Included

### Task 1 – Calculator Application
- Basic calculator with arithmetic operations
- Built using Kotlin and ConstraintLayout

### Task 2 – Sensor Data Reader Application
- Reads real-time data from all available device sensors
- Dynamic sensor selection
- Battery-efficient implementation

### Task 3 – Developer Evangelist Blog
- Step-by-step article explaining how to build the Sensor Reader app
- Written in Markdown format
